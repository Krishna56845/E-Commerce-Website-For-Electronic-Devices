# AI Coding Agent Instructions

This document provides guidance for AI agents working on this e-commerce platform codebase.

## Project Overview

This is a full-stack e-commerce application built with modern web technologies. It includes:
- Product browsing and search
- Shopping cart functionality
- User authentication
- Checkout process
- Backend API

## Key Directories

- `backend/` - Node.js Express API
- `src/` - React frontend
- `public/` - Static assets

## Common Tasks

1. **Add new features** - Follow existing patterns in components and API routes
2. **Fix bugs** - Check backend routes and frontend components
3. **Update dependencies** - Run `npm install` in both directories

## Development Workflow

1. Make changes
2. Test locally
3. Run tests: `npm test`
4. Deploy to your chosen platform
